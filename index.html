Название теста: Тест по базовому тестированию

Описание:

«Тест по базовому тестированию» представляет собой интерактивный тест, который позволяет проверить ваши знания в области тестирования программного обеспечения. Этот тест содержит четыре вопроса с вариантами ответов и балльную оценку системы.
